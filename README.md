# Frank-Wolfe-Optimization-for-Graph-Problems
Scalable optimization algorithms for the Maximum Clique Problem in large graphs, implementing Frank-Wolfe variants including Pairwise and Away-Step to optimize speed, stability, and solution quality on real-world benchmarks

## Co-Author
Liliia Ivanovskaia


## Description

Developed and evaluated scalable optimization algorithms for complex graph problems using real-world benchmark data. Implemented multiple variants of the **Frank-Wolfe algorithm** (including Pairwise and Away-Step) to efficiently approximate solutions for the **Maximum Clique Problem**, with a focus on performance, scalability, and solution quality.

### Key Results
- Significantly improved solution quality through tailored regularization techniques
- Analyzed performance trade-offs between speed, stability, and accuracy across algorithm variants
- Achieved strong results on large-scale graphs (up to 400 nodes / 78k edges), with best solutions reaching clique sizes of 54
